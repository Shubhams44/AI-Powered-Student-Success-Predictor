# AI Powered Student Success Predictor

## Overview

This machine learning project predicts whether a student is likely to succeed based on academic performance indicators.

## Features

- Success Prediction
- Performance Analysis
- Interactive Dashboard
- Real-time Predictions

## Technologies Used

- Python
- Pandas
- Scikit-Learn
- Streamlit

## How to Run

python -m streamlit run app.py
